CLASS
System Effect Network
System has builtin effect that is sending and receiving info with any.

Network destination is identified with network port.

Network port has ideally 3 60 bits Ints.

Network destination can be a network any of a module execution of a system. The first and second 60 bits Ints identify the system. The third 60 bits Int identifies the opening network any.

Network does any effect by sending and receiving info with anys by the network ports of the anys.

Network has ideally minimum speed of sending and receiving bytes that is 4 kilo bytes per tick.
