System Effect Type
System has builtin effect that is receive typing from system user.

Type is ideally taking input from 2 hands of user. Type is ideally used with each hand covers 1 half of the type board.

Type is ideally layouted with its buttons positions at symmetrical left and right. Each button has the other matching button on the other side.

Type is ideally used with its position at the front of, horizontally center of, and vertically below system video position.

Type buttons take ideally small force to press down. Type buttons take ideally small force to keep pressed down.

Type is easy to be used continuously for long duration of up to half a day for 629 days.

Letters on Type buttons are capital letters. Lower case letters are not shown on Type. Letters on Type buttons are differentiated from lower case letters.
